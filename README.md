## eopkgTui

   ######  Simple text based eopkg command(s) User Interface.

## Pre-request :

    
    sudo eopkg it fzf


  1. Download/clone the repo or copy the script 
  2. Change the file permission (chmod +x eopkgTUI.sh)
  3. Run ./eopkgTUI.sh
