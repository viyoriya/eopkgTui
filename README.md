## eopkgTui

   #####  Simple text based eopkg command(s) User Interface (Until sol/ferryd release)

## Pre-request :

    
  ####  sudo eopkg it fzf


  #####  Download/clone the repo/script 
  #####  change the file permission (chmod +x eopkgTUI.sh)
